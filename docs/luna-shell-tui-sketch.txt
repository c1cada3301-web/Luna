================================================================================
  Решение (зафиксировано)
================================================================================

  Glyph:     🌙 emoji — welcome, login, status header
  Prompt:    ◐ luna:path$  (уже в PS1, без emoji)
  Thinking:  мигание символов 🌙 ◐ ◑ ◒  или braille spinner ⠋⠙⠹…
  Карточка:  рамка ┌─┐│└┘ + 🌙 слева + Tips / What's new справа
  Шрифт:     JetBrains Mono 15 (как Cursor); в VB tty пока системный mono

  Не используем: █ pixel-art, ASCII-кривые .-"'-, braille-«шары» по умолчанию.

================================================================================
  Agent «thinking» — мигание символов
================================================================================

  Вариант 1 — фазы луны (4 символа, ~0.35s):

    🌙 → ◐ → ◑ → ◒ → 🌙 …

  Вариант 2 — braille spinner (1 символ, классика CLI):

    ⠋ ⠙ ⠹ ⠸ ⠼ ⠴ ⠦ ⠧ ⠇ ⠏

  Строка под prompt:

    ◑ thinking…

  Псевдокод:

    phases='🌙 ◐ ◑ ◒'
    while agent_busy; do
      for p in $phases; do
        printf '\r  %s  ' "$p"
        sleep 0.35
      done
    done

================================================================================
  Шрифт — «как в Cursor»
================================================================================

  В ISO **шрифт пока не зашит** — VirtualBox рисует стандартный bitmap mono,
  поэтому welcome в VB выглядит иначе, чем в Cursor.

  Целевой шрифт Luna Shell (как у тебя в Cursor):

    JetBrains Mono · 15px · ligatures on

  ┌──────────────────┬─────────────────────────────────────────────────┐
  │ Где смотришь Luna │ Как получить JetBrains Mono                      │
  ├──────────────────┼─────────────────────────────────────────────────┤
  │ VirtualBox tty   │ Пока системный mono — не совпадает с Cursor      │
  │ SSH luna@vm-ip   │ iTerm/Terminal → JetBrains Mono 15, ligatures   │
  │ Cursor terminal  │ Уже JetBrains Mono — welcome совпадёт           │
  │ Будущее Wayland  │ foot + JetBrainsMono Nerd Font в образе         │
  └──────────────────┴─────────────────────────────────────────────────┘

  Запасные: Geist Mono, SF Mono, IBM Plex Mono.

  Для dev: тестируй welcome через `ssh luna@10.0.2.15` — тот же шрифт, что в IDE.
