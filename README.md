**Фаза:** подготовка  
**Следующий шаг:** Milestone 0 — собрать минимальный ISO и загрузить его в VirtualBox/QEMU

## Быстрые решения (на сегодня)

- **Подход:** свой Linux-дистрибутив, не kernel с нуля
- **База:** Alpine Linux (minimal, OpenRC, `apk`, ~130 MB base)
- **Хост для сборки:** macOS + Docker (или Linux VM)
- **Тест:** VirtualBox и/или QEMU
- **Языки на этом этапе:** shell, конфиги; Rust/Go — позже для userspace-утилит
